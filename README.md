# geocoding-sheet

An addon for Google Sheets that provides the ability to geocode location data

## Setup and tests

### Prerequisites

* Bash shell
* Node
* Yarn

### Install

`yarn install`

### Test

`yarn test`

## Manage GCP artifacts

### Build GCP artifacts

`yarn build`

### 

### Push GCP artifacts

`yarn clasp-push`

### Build, version and deploy GCP artifacts

`yarn clasp-deploy`
